Step 0 - Allowing Email to Send These Messages:

To prevent scams, email won't let you send emails automatically, we can get around this by making it semiautomatic instead

This still requires some changes to be done. First go to chrome://settings/handlers (this couldn't be link due to chrome security)

Find mail.google.com and check if its already set to default and that sites can ask to handle protocols is checked. If not do so for both.

Make sure prior to use that you are signed into the CYC email account. For extra security it is recommended to choose a special background for the email as it will carry over to the temporary display. You can also try the test email button on the main page

Before sending the bulk emails, you should try the Test Email button. If you enter your email it doesn't open a new tab with the email template, go to where the URL address is, click the informtion icon (the small i in a circle) and then enable Pop-Ups and Redirects.

Step 1 - Formatting the Template:

In order for the emailer to find and replace the necessary information, special marks need to be added to the template

To do so add a dollar sign followed by the text in curly braces to indicate a fill in as seen below

Also don't forget to put the email subject first with the following formatting


ORIGINAL TEMPLATE

Subject: Giftcards

Hello (PLACE),

We, as the CYC, were hoping to reach out regarding getting $(MONEY) giftcards for our raffle.

Thanks,

CYC


FIXED TEMPLATE

SUBJECT: Giftcards

Hello ${PLACE},

We, as the CYC, were hoping to reach out regarding getting $${MONEY} giftcards for our raffle.

Thanks,

CYC

Step 2 - Creating Formatted Spreadsheet:

For every fill in present in the spreadsheet, it is required to have a column dedicated for that fill in. The spreadsheet should be in Microsoft Excel (.xlsx) format which can be downloaded from Google Sheets

Additionaly, there should be two extra columns - one for the email and one for if it has already been sent

Using the previous template, you can see this example

Example


The main things to note are:

1. The email and send columns are first

2. The information columns headers exactly match the format of the fill in spaces in the template

3. Some of the sent column have values. These rows will not be considered for emailing (Only KFC, Starbucks, and Pizza Hut will be emailed)

3a. Note: The sent column doesn't need to have dates, anything can be put in the cell to omit it from being sent

Step 3 - Sending the Emails:

Make sure you are signed into the account you want to send the emails from

This program is specifically designed to look for the CYC email when attempting to send the email

Finally, before attemping the bulk send it is very much recommended to try the Test Email button to flush out any problems.
